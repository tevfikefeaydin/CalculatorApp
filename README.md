# CalculatorApp

A simple WPF calculator with a modern dark-themed UI.

## Features
- Dark theme with color-coded buttons (numbers, operators, functions)
- Rounded buttons with hover/press feedback
- Running expression preview above the result
- Backspace (⌫), sign toggle (±), and percent (%)

## Run
Requires the .NET SDK with the WPF workload on Windows:

```
dotnet run --project CalculatorApp
```
